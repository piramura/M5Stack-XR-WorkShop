# Unity PCのIPアドレス確認手順

M5StackとUnity PCを同じWi-Fiへ接続してから、Unity PCのIPv4アドレスを確認します。

## macOS

### システム設定で確認する

1. アップルメニューから「システム設定」を開く
2. 「ネットワーク」を開く
3. 接続中のWi-Fiを選択して「詳細」を開く
4. 「TCP/IP」に表示されるIPアドレスを確認する

### ターミナルで確認する

Wi-Fiが`en0`の場合は次のコマンドを実行します。

```bash
ipconfig getifaddr en0
```

何も表示されない場合は、システム設定から確認してください。

## Windows

### 設定画面で確認する

1. 「設定」を開く
2. 「ネットワークとインターネット」から「Wi-Fi」を開く
3. 接続中のネットワークのプロパティを開く
4. 「IPv4 アドレス」を確認する

### コマンドプロンプトで確認する

次のコマンドを実行します。

```text
ipconfig
```

「ワイヤレス LAN アダプター Wi-Fi」に表示される「IPv4 アドレス」を確認します。

## M5Stackの送信先を設定する

確認したIPv4アドレス全体を`include/wifi_config.h`の`OSC_DESTINATIONS`へ設定します。

```cpp
static constexpr OscDestination OSC_DESTINATIONS[] = {
    {"PC A", "172.16.80.100"},
    {"PC B", "172.16.80.101"},
    // {"PC C", "172.16.80.102"},
};
```

左側の名前はM5Stackの選択画面に表示されます。使用しない行はコメントアウトできます。IPアドレスの末尾だけでなく、表示されたアドレス全体を入力してください。
